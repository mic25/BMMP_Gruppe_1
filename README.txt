Hey!

Ihr bekommt den Link zum Repository.
Ihr müsst euch bei github.com einen Account anlegen und am Besten Github auch runterlasen (macht das organisieren einfacher).
Dann ladet ihr das Repository runter, sodass ihr das Verzeichnis auf eurem PC lokal gespeichert habt.
Ihr könnt dann lokal Änderungen vornehmen und diese dann mit Github für die Gruppe hochladen. Wenn ihr Github öffnet, werden automatisch geänderte Dateien im Repository angezeigt. Ihr könnt dann auswählen, ob ihr die ganze Datei oder nur einzelne Zeilen als Änderung hochladen wollt.

Viel Spaß!
